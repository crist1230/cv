  • Horizontal Rule Tags
    • Creates a horizontal line on the page
    • To implement use "hr"
    • Attributes:
      • size="#" (sets the size in pixel of the line)
      • noshade (removes shading from the line)

  • Image Tags
    • Applies an image onto the page
    • To implement use "img"
    • Attributes:
      • src="#" (tells the computer where to get the image)
      • alt="#" (if the page cannot load the image this tells the page what words to print)
        • NOTE: be sure to use words that describe what the image is so seach engines are able to more easily identify what the website contains
      • width="#" (changes width in pixels of the image)
      • height="#" (changes height in pixels of the image)

  • Input Tags
    • NOTE: for information on when to use this tag please visit the Non-Closing Tags page
    • Tells a page what to implement onto a form
    • To implement use "input"
    • Attributes:
      • "type" describes what type of form the page recieves
        • Available Types:
          • "button"
          • "checkbox"
          • "color"
          • "date"
          • "email"
            • NOTE: this type is really cool in that it checks the text put into the box to verify it is an actual email
          • "file"
          • "password"
          • "radio" this is a little confusing but it's for selecting only one option when you have multiple
          • "search"
          • "submit"
          • "text"
