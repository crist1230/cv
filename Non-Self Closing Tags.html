  1. Anchor Tags
    • Connects text to other pages
    • To implement use "a"
    • Attributes:
      • href="#" (put in the url of the page you would like to direct to)
      • NOTE: before you close the tag put what text you want the hyperlink to be attached to

  3. Form Tags
    • Creates a form on the page in which visitors can enter information
    • To implement use "form"
    • Connected Elements:
      • "label" tells the page what to label a certain attribute
      • "input" tells the page what the form should have in it
        • NOTE: for more information regarding the input element and its attributes please visit the Input Tag Section on the Self-Closing Tags page
      • "textarea" implements a box where visitors can put in text
        • Attributes:
          • "rows" how many rows in pixels the textbox has
          • "cols" how many columns in pixels the textbox has
    • Attributes:
      • "enctype" tells the form how to encode the message
        • Types:
          • "text/plain" encoding in plain text
      • "action" tells what the form should do
        • Types:
          • "mailto:'#'" emails the form to a specified address
      • "name" gives what the visitor is entering into a form a label

  5. Table Tags
    • Creates a table on a page
    • To implement use "table"
    • Connected Elements:
      • NOTE: all are non-self closing elements
      • "thead" inserts a table header (where the titles go)
        • when adding data to the head, use the element "th" which automatically centers and bolds the heading text with the table data
      • "tbody" inserts a table body (where the data goes)
      • "tfoot" inserts a table footer (where notes may go)
      • "tr" inserts a table row
      • "td" where you insert your data
    • Attributes:
      • "cellspacing='#'" spaces the cells a certain amount of pixels from each other
      • "border" places a border around the cells
